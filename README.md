# Urban Craves Showcase (Static Site)

This repository now contains a **fully static website** with no framework and no build step.

## Stack
- HTML (`index.html`)
- CSS (`styles.css`)
- Vanilla JS (`scripts.js`)
- Static images (`assets/`)

## Local preview
Open `index.html` directly in a browser, or run any static file server from the repository root.

## GitHub Pages
This repository is ready for direct root deployment on GitHub Pages.
